# 📊 Relatório Mensal

**Sistema Web para Gerenciamento de Relatórios Mensais de Atividades**

## ✨ Visão Geral

Aplicação web desenvolvida para a InterOp, destinada ao registro,
organização e geração de relatórios mensais de atividades internas.\
Funciona totalmente de forma local, **sem necessidade de instalação ou
servidor**.

## 🔧 Funcionalidades

### 📋 Identificação do Colaborador

-   Registro de informações pessoais (nome, cargo, e-mail)\
-   Seleção do mês e ano de referência

### 📝 Gerenciamento de Atividades

-   Adicionar, editar e excluir atividades\
-   Categorização por área demandante\
-   Controle de horas trabalhadas\
-   Status: **Em andamento**, **Concluído**, **Pendente**, **Cancelada**

### 📊 Síntese Mensal Automática

-   Total de atividades\
-   Soma automática de horas\
-   Campo para principais resultados

### 💡 Sugestões e Melhorias

-   Estrutura organizada: **Situação atual → Processo sugerido →
    Resultado esperado**

### 🔄 Funcionalidades Avançadas

-   Salvamento automático no navegador\
-   Geração de PDF via impressão\
-   Interface responsiva (desktop e mobile)\
-   Modal de confirmação para exclusões

## 🚀 Como Usar

### 1. 🔓 Acesso

Abra o arquivo **index.html** em qualquer navegador moderno.

### 2. 🧾 Preenchimento

-   Preencha seus dados\
-   Adicione atividades\
-   Utilize "Calcular Totais"\
-   Preencha Síntese e Sugestões

### 3. ⚙️ Gerenciamento

-   ✏️ Editar\
-   🗑️ Excluir\
-   🔄 Salvamento automático

### 4. 📄 Exportação (PDF)

-   Clique em "Salvar como PDF"\
-   Ajuste configurações do navegador
